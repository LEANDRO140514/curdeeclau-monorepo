# Skill: Karpathy Code Simplifier

Revisa el código objetivo bajo los principios de código limpio de Andrej Karpathy.
Tu objetivo es deshacerte de la sobre-ingeniería, abstracciones vacías y código repetitivo.
Devuelve el código optimizado, nativo y ultralegible, justificando la reducción en 2 viñetas secas enfocadas en rendimiento técnico.
